What is Bootstrap-Swapbox?

A bootstrap-extension that converts a standard select box into a select box with a twist.

The twist is that while other select boxes leave the dropdown options the same after you choose one, this select box will hide the selected option from the list.

I know, not that exciting... but to see similar functionality in the wild, checkout the News Feed filter button on the Facebook Mobile App on iOS. This is what I was going for.

Simple Usage / Example

<select id="swapbox-demo-1" name="choice-toggle">
    <option data-icon-class="fa fa-book" value="all">All Upcoming Events</option>
    <option data-icon-class="fa fa-calendar" value="live">Upcoming Live Events</option>
    <option data-icon-class="fa fa-cloud" value="online">Online Courses</option>
</select>

<script>
    $(function() {
        $("#swapbox-demo-1").swapbox();
    });
</script>

Dynamic Append / Prepend Overrides

<select id="swapbox-demo-2" name="choice-toggle">
    <option data-icon-class="fa fa-book" value="all">All Upcoming Events</option>
    <option data-icon-class="fa fa-calendar" value="live">Upcoming Live Events</option>
    <option data-icon-class="fa fa-cloud" value="online">Online Courses</option>
</select>

<script>
    $(function() {
        $("#swapbox-demo-2").swapbox({
            itemPrepend: function(item) {
                return "<a href='#'><span class='" + item.$option.attr('data-icon-class') + "'></span> ";
            },
            itemAppend: function() {
                return "</a>";
            }
        });
    });
</script>
